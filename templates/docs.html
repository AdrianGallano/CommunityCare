CommunityCare API Documentation

Endpoints:

Authentication and Authorization:

Authentication is required for certain endpoints. Use the following endpoints to obtain and refresh JWT tokens:

  • POST /api/token/ - Obtain JWT token
  • POST /api/token/refresh/ - Refresh JWT token

Searching:

  • ?search=value

Ordering:

  • ?ordering=value

Pagination:

  • ?page_size=value&page=value

  • page_size = resources per page
  • page = page number (offset)

Resources:

Location:

  • GET /api/locations - List all locations
  • POST /api/locations - Create a new location
  • GET /api/locations/{id} - Retrieve a single location
  • PUT /api/locations/{id} - Update a location
  • PATCH /api/locations/{id} - Partially update a location
  • DELETE /api/locations/{id} - Delete a location
